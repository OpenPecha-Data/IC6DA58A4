irdiṣṭāyāṃ prajñāpāramitāyāṃ na kevalaṃ catvāro lokapālā mahārājāna upasaṃkramitavyaṃ maṃsyante, aham api bhagavaṃs taṃ 
bodhisattvaṃ mahāsattvam upasaṃkramiṣyāmi, kaḥ punar vādas tadanye devaputrāḥ |tathāgatair api so ’rhadbhiḥ samyaksaṃbud
